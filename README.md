## 前提

    1.你的手机必须已经root
    
    2.手机和电脑必须在同一个局域网

    3.手机已经完成服务端设置，参考https://blog.csdn.net/zjq592767809/article/details/127825710


## 如何使用

    1.确保前提已经完成
    
    2.安装项目依赖pip install - r requirements.txt
    
    3.把你手机的局域网ip与端口修改到main.py的入口函数中
    
    4.运行python main.py
